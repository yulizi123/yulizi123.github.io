3. Scipy >= 0.11

直接复制粘贴下面的命令, 然后在 terminal 当中运行. 如果要使用 GPU 加速运算, 要确保在你的电脑上有 NVIDIA 的 GPU 显卡, 而且能够使用 CUDA 模块.

```shell
# python 2+ 版本
pip install theano

# python 3+ 版本
pip3 install theano
```

如果遇到权限问题, 请在 `pip` 前加一个 `sudo` 如: `sudo pip install theano`.
