一个一维的List是线性的List，多维List是一个平面的List：

```python
a = [1,2,3,4,5] # 一行五列

multi_dim_a = [[1,2,3],
			   [2,3,4],
			   [3,4,5]] # 三行三列
```


{% include assign-heading.html %}

在上面定义的List中进行搜索：

```python
print(a[1])
# 2

print(multi_dim_a[0][1])
# 2
```

用行数和列数来定位list中的值。这里用的是二维的列表，但可以有更多的维度。
