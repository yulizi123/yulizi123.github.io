`python`可以直接运算数字，也可以加`print` 进行运算.

```python
>>> 1+1
2
>>> 2-1
1
>>> 2*3
6
>>> 4/3
1.3333333333333333
```

{% include assign-heading.html %}

`python`当中`^`符号，区别于`Matlab`，在`python`中，`^`用两个`**`表示，如3的平方为`3**2` , `**3`表示立方，`**4`表示4次方，依次类推

```python
>>> 3**2   # **2 表示2次方
""""
9
""""
>>> 3**3   #  **3 表示3次方
""""
27
""""
>>> 3**4
""""
81
""""

```
